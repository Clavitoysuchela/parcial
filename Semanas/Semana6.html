Semana 6: Componentes en React con JSX

🧩 ¿Qué es JSX?

JSX (JavaScript XML) es una sintaxis que permite escribir HTML dentro de JavaScript. Es una forma más legible y eficiente de crear elementos React sin usar React.createElement().

  • Permite combinar lógica y presentación en componentes.
  • Puede incluir expresiones JavaScript entre { }.
  • Compilado por Babel a JavaScript puro.

🧱 Componentes en React

React permite dividir la interfaz en componentes reutilizables y escalables. Se pueden construir como funciones o clases (aunque actualmente se prefiere la versión funcional).

  • Componentes funcionales con hooks.
  • Uso de props para pasar información.
  • Modularidad y reutilización de UI.

🧪 Práctica 3: Lista de Usuarios

En esta práctica se creó un componente llamado ListaUsuarios que recibe un array de objetos y los renderiza dinámicamente usando JSX.

// ListaUsuarios.jsx
import React from 'react';

function ListaUsuarios(props) {
  return (
    <div>
      <h3>Usuarios Registrados</h3>
      <ul>
        {props.usuarios.map((u, i) => (
          <li key={i}>{u.nombre} - {u.email}</li>
        ))}
      </ul>
    </div>
  );
}

export default ListaUsuarios;

Se utilizó el componente con datos simulados como:

const usuarios = [
  { nombre: "Ana", email: "ana@mail.com" },
  { nombre: "Luis", email: "luis@mail.com" }
];
