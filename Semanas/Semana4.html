🧱 Introducción a TypeScript

TypeScript es un superset de JavaScript que permite trabajar con tipos estáticos y clases modernas, facilitando la escalabilidad en proyectos grandes.

  • Compilación de .ts a .js.
  • Definición de tipos y clases.
  • Interfaces y tipado de funciones.

🎨 Canvas con JavaScript

El elemento <canvas> permite dibujar directamente en el navegador. Usamos su API para crear gráficos dinámicos.

⬆️ Ejemplo simple: dibujo con líneas y relleno

const canvas = document.getElementById("myCanvas");
const ctx = canvas.getContext("2d");

ctx.fillStyle = "#60a5fa";
ctx.fillRect(20, 20, 100, 50);
ctx.strokeStyle = "#1e40af";
ctx.lineWidth = 4;
ctx.strokeRect(20, 20, 100, 50);
