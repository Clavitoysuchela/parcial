Semana 10: API RESTful para Gestión de Docentes

Resumen del Proyecto

Esta semana se desarrolló una API RESTful completa para la gestión de docentes utilizando principios de arquitectura REST, permitiendo realizar operaciones CRUD (crear, leer, actualizar y eliminar) sobre los datos de los profesores. El backend se implementó en Spring Boot con conexión a una base de datos relacional.

📌 Endpoints CRUD

Rutas organizadas por controladores REST que permiten listar, registrar y editar docentes.

🛠️ Validaciones

Uso de anotaciones de validación con Bean Validation (javax.validation).

🔒 Seguridad

Diseñado para integrarse fácilmente con Spring Security en futuras versiones.

Tecnologías y Herramientas Utilizadas

  • 🌐 Spring Boot
  • 🗄️ MySQL
  • 📦 Maven
  • 📨 Postman
  • 📁 JPA (Hibernate)
